Few Words About The Project

Description

Esport is rapidly increasing in popularity and it's becoming more and more serious, there even are discussion about including esport as a non official competition at the Tokyo Olympic games in 2021. Eventough some people might only hear about Esport recently it has been there for a while there already was tournament in 1998! We wondered how that growth in notoriety has impacted the number tournament and the cash prize. It will also be interesting to observe which kind of game is the most popular and have the more tournament and the biggest cashprize. We can also analyze if a game stay popular a long time or not it might be popular once it is released but die a few years later and we also interest ourselves in the impact of covid-19 on Esport as some would guess that it's only compute game you can do it at home so there should not be any impact we will see if that's true or not. We can then focus more on the player to see if it's really possible to live from tournament and see which player earned the most!
